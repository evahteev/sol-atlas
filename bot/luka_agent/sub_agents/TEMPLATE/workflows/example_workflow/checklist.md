# Example Workflow - Validation Checklist

Use this checklist to validate the workflow output before finalizing.

---

## Content Quality

- [ ] **Completeness:** All required sections are present
- [ ] **Accuracy:** Information is factually correct
- [ ] **Clarity:** Content is clear and easy to understand
- [ ] **Consistency:** Terminology is consistent throughout
- [ ] **Grammar:** No spelling or grammatical errors

---

## Structure

- [ ] **Executive Summary:** Concise overview of main points (1-2 paragraphs)
- [ ] **Main Sections:** All key topics covered with appropriate depth
- [ ] **Logical Flow:** Sections follow a logical progression
- [ ] **Headers:** Clear and descriptive section headers
- [ ] **Formatting:** Consistent markdown formatting

---

## Objective Alignment

- [ ] **User Goal:** Output addresses the stated objective
- [ ] **Constraints:** All constraints have been respected
- [ ] **Desired Outcome:** Result matches the desired outcome
- [ ] **Scope:** Content stays within defined scope

---

## Actionability

- [ ] **Key Findings:** Clear and well-supported findings
- [ ] **Recommendations:** Specific, actionable recommendations
- [ ] **Next Steps:** Clear action items with owners (if applicable)
- [ ] **Timeline:** Realistic timeline for next steps (if applicable)

---

## Supporting Information

- [ ] **Data Sources:** All data sources properly cited
- [ ] **References:** External references included where needed
- [ ] **Examples:** Concrete examples provided where helpful
- [ ] **Appendix:** Supporting materials included

---

## Format & Style

- [ ] **Length:** Appropriate length for the topic
- [ ] **Tone:** Matches the intended audience
- [ ] **Visual:** Easy to scan (bullets, headers, whitespace)
- [ ] **Accessibility:** Readable by target audience

---

## Final Checks

- [ ] **Metadata:** Date, author, version information complete
- [ ] **File Name:** Follows naming convention
- [ ] **File Location:** Saved to correct directory
- [ ] **Backup:** Draft versions backed up if needed
